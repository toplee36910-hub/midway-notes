## 这是什么

"半路手记"是一个技术实践博客，记录一个没有编程背景的人如何用 AI 工具完成真实项目。不是教程，不是 demo，是真正上线、真正有人用的东西。

作者安远，五十岁开始用 AI 做事。做过概念音乐专辑，从零搭过网站，做过声纹克隆有声书——全程 AI 协作，手写代码为零。
